## Engage AI

A data-driven learning analytics platform that collects student study behaviour from multiple sources—such as Azure databases, Panopto video repositories, and learning management systems (e.g. Canvas and Moodle).
It provides personalised learning reminders for assignments and video-watching tasks, engagement metrics, and insights to support students throughout their learning journey.
The platform also enables academic staff to monitor student engagement levels and identify at-risk students early, allowing timely intervention and support.

- **Tech**: AWS (Lambda, DataZone, S3), Python, FastAPI, Azure Synapse Analytics, Azure SQL Database
- **Status**: In Progress

## Evaluation Metrics Dashboard

- **Tech**: AWS (Lambda, DataZone, S3), Python, FastAPI, Azure Synapse Analytics, Azure SQL Database, VuePress
